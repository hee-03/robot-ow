파지: createConstraint(JOINT_FIXED)로 안정적 물체 고정

페이즈	동작	목표 좌표
Phase 6 TRANSIT	목적지 위로 수평 이동	[0.5, 0.3, 0.35]
Phase 7 PLACE_DESCEND	바닥까지 하강	[0.5, 0.3, 0.06]
Phase 8 RELEASE	제약 해제 → 그리퍼 개방 → 후퇴	-

실행 흐름:
1. PyBullet GUI 열림 → 3개 물체 + 로봇 표시
2. 자연어 입력 대기
   >> 빨간 네모를 0.3 0.2 0.025 로 빠르게 옮겨줘
3. LLM 파싱 → object, destination, speed_level
4. 360° 스캔 → OWL-ViT로 해당 물체 탐색
5. 목적지 주변 12cm 이내 다른 물체 감지 시
   → 해당 물체를 x+15cm, y+15cm로 먼저 이동
6. 목표 물체 pick & place → 지정 좌표에 내려놓기
